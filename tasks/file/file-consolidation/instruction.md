# File Consolidation

Find all profile.txt files in /workspace/users/ directories. Each profile.txt contains Email, Role, and Action Items lines. Read each profile and create a CSV file named 'users_summary.csv' in /workspace with columns: name, email, role, action_count. The name should be the directory name (e.g., 'Alice Johnson'). Sort the rows by action_count in descending order (highest first).
