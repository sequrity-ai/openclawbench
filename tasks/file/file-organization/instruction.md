# File Organization

Read /workspace/sales_data.csv. Create a directory called 'by_region/' under /workspace. For each unique region in the data, create a sub-directory named after the region (e.g., 'by_region/North/', 'by_region/South/'). In each region directory, create a file called 'sales.csv' containing ONLY the rows for that region, with the same CSV header (product,quantity,price,region).
